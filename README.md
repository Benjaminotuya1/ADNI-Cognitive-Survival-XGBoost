# ADNI-Cognitive-Survival-XGBoost
Predicting cognitive decline and Alzheimer's progression using longitudinal ADNI clinical data and XGBoost survival modeling.
Project: Predicting Cognitive Decline using ADNI Clinical Data
Phase 1: Engineered a longitudinal dataset from 16,481 raw clinical visits. Aligned fragmented metabolomic, volumetric MRI, and demographic timelines into a unified cohort of 11,518 patient visits for XGBoost survival modeling.
